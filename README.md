# 6 颗 IN-12 辉光管时钟方案

> 来自 B 站：[【源码+PCB开放】制作一台属于自己的辉光钟](https://www.bilibili.com/video/BV1NJ411T7A1)

手搓 6 颗 IN-12 辉光管的时钟，TPIC6B595 驱动 + MAX1771 高压升压，ESP32 主控。

---

## 系统架构

```
12V DC
  │
  ├──→ [MAX1771 Boost] ──→ 170V HV ──→ 辉光管阳极
  │
  └──→ [AMS1117-5V] ──→ [AMS1117-3.3V] ──→ ESP32
                                                │
                          ┌─────────────────────┤
                          │                     │
                    [TPIC6B595 × 9]         [DS3231 RTC]
                     SPI 级联 72 路          I²C 时钟
                          │
                     [IN-12 × 6]
                     6 管 × 12 脚
```

## 工作原理

辉光管（Nixie Tube）是充氖气的冷阴极辉光放电管。每个管子内部有 10 个数字阴极（0–9）+ 小数点，当某阴极与阳极之间施加约 **170V 直流电压** 时，该阴极周围的氖气被电离，发出橙红色辉光。

6 颗管子共需控制 **72 个阴极**（6 × 12），单片机的 GPIO 远远不够。

通过 **3 根 SPI 线串联 9 片 TPIC6B595 移位寄存器**，每片可控制 8 路高压开漏输出，级联后共 72 路，分别控制每颗管子的每个数字。

## 设计要点

- **170V 高压安全**：操作前务必断电等待电容放电（≥30 秒），单手操作原则
- **限流电阻**：每管阳极串联 10KΩ，限制辉光电流约 2–3mA
- **防阴极中毒**：固件中定期轮显 0–9，防止长期不用的数字发暗
- **NTP 对时**：ESP32 通过 WiFi 自动获取网络时间，掉电后由 DS3231 电池保持
